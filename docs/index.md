# Libro Digital de Programación

Este repositorio publica el curso como un libro digital orientado a competencias. Cada unidad funciona como un capítulo y comienza con una guía teórica que enmarca la competencia a desarrollar antes de entrar a las sesiones de trabajo.

## Qué encontrará el estudiante

- El sílabo del curso.
- Una guía de apoyo para pasar de algoritmo a código.
- Tres capítulos organizados por competencias.
- Una guía teórica al inicio de cada capítulo.
- Las 16 sesiones de trabajo y sus evaluaciones.
- Ejemplos y ejercicios presentados como material de lectura y práctica.

## Importante

Este sitio está pensado como libro digital. El contenido se muestra como páginas estáticas y no ejecuta código en el navegador.

## Recomendación de uso

En cada capítulo, lea primero la guía teórica para entender la competencia central, luego avance por las sesiones en orden y cierre con la evaluación correspondiente. La secuencia está diseñada para progresar desde problemas básicos hasta soluciones modulares con persistencia elemental.
