# Libro Digital de Programación

Este repositorio publica los cuadernos del curso como un libro digital estático en GitHub Pages.

## Qué encontrará el estudiante

- El sílabo del curso.
- Una guía de apoyo para pasar de algoritmo a código.
- Las 12 sesiones organizadas por unidades.
- Ejemplos y ejercicios presentados como material de lectura.

## Importante

Este sitio está pensado como libro digital. El contenido se muestra como páginas estáticas y no ejecuta código en el navegador.

## Estructura

- Unidad 1: fundamentos para pensar y programar.
- Unidad 2: automatizar y organizar programas.

## Recomendación de uso

Lee cada sesión en orden, revisa los ejemplos y luego desarrolla los ejercicios en tu propio entorno de Python o Jupyter.
